## User Roles

1. Super Admin
2. Director
3. Operations Manager
4. HR
5. Sales Manager
6. Team Lead
7. Employee
8. Freelancer
9. Client

## Permission Overview

| Role | Access Level |
|---|---|
| Super Admin | Full system access |
| Director | Company dashboard, reports, finance overview |
| Operations Manager | Projects, teams, tickets, approvals |
| HR | Employees, attendance, leave, recruitment |
| Sales Manager | Leads, CRM, proposals |
| Team Lead | Tasks, tickets, approvals, team worklogs |
| Employee | Assigned tasks, attendance, leave |
| Freelancer | Assigned work only |
| Client | Client portal only |
