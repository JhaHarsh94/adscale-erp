# AdScale One ERP

AdScale One ERP is a complete agency operating system designed for managing employees, departments, CRM, projects, tasks, tickets, attendance, approvals, files, chat, meetings, HRMS, payroll, finance, ads, SEO, automation, AI, and future SaaS expansion.

## Project Vision

To build one centralized platform for AdScale Media where all departments, employees, clients, projects, tasks, tickets, reports, communication, and automation can be managed from a single system.

## Tech Stack

Frontend: Next.js, React, TypeScript, Tailwind CSS  
Backend: Node.js, Express.js, TypeScript  
Database: PostgreSQL  
ORM: Prisma  
Authentication: JWT + Bcrypt  
Realtime: Socket.IO  
Queue/Cache: Redis  
Storage: Cloudinary / S3  
Meetings: Jitsi Meet  
Deployment: Vercel, Render, Supabase PostgreSQL
